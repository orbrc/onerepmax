# One Rep Max (1RM) Calculator

This project is a tool designed to estimate your One Rep Max (1RM)—the maximum weight you can lift for a single repetition in a given exercise.

Instead of relying on a single formula, it enhances accuracy by:
1. Taking the input: The user provides the weight they lifted and the number of repetitions performed (e.g., 100 kg for 5 reps).
2. Calculating with three standard formulas: It computes the estimated 1RM using three popular strength equations:
 - Epley
 - Brzycki
 - Lander
3. Averaging the result: The final output is the arithmetic mean (average) of these three calculations.

This approach provides a more balanced and reliable estimate than any single formula alone, helping athletes and lifters accurately track their strength progress and plan their training loads.
